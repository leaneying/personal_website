WanderingMelody

GitHub Live Demo

Description

Wandering Melody is a personalized music recommendation engine designed to help users discover global sounds that resonate with their emotions and environment. By combining user inputs such as mood, age, location, and weather, the platform generates curated soundtracks that reflect both the emotional tone and cultural identity of different regions.

Date: January - May 2025
Company/Organization: Cornell University
Role: Full-stack developer

Technologies Used

Python Flask HTML/CSS NLTK Scikit-learn Pandas NumPy GitHub

Key Features

  • Mood-based filtering and weather-aware scoring
  • Location-driven exploration for culturally relevant music powered by Google Maps reviews
  • Latent topic modeling using natural language processing (NLP)
  • Popularity ranking powered by Spotify data
