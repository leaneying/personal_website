Description

The OKB Hope Foundation works to transform healthcare provision in Ghana through collaborative education, cutting-edge research, and patient-centered care. The OKB Hope Foundation website is platform aimed to streamline the process to connect users with psychiatrists based on personal preferences, offering virtual appointments via chats and videos across the globe.

Date: January - August 2024
Company/Organization: Hack4Impact
Role: Full-stack developer

Technologies Used

JavaScript/TypeScript Firebase React Figma Node.js GitHub

Key Features

  • Seamless user registration and authentication via Firebase
  • Editable user profiles for personalized experiences
  • Dedicated professional profiles tailored for doctors
  • "Discover Professionals" page for easy provider search and access
  • Secure, real-time messaging powered by Firebase
  • Comprehensive data analytics dashboard for insights and monitoring
