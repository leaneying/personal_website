The Wash Foundation

Description

The WASH Foundation (formerly Clean the World Foundation) is a global nonprofit working to improve public health through increased access to clean water, sanitation, and hygiene. In collaboration with the foundation, we developed a data visualization web application to support their fog net initiatives in Lima, Peru: passive systems that harvest atmospheric moisture in areas with limited water access. The platform helps stakeholders evaluate and scale fog net deployments by visualizing critical data points and performance metrics. (Note: not currently deployed)

Date: September - December 2023
Company/Organization: Hack4Impact
Role: Full-stack developer

Technologies Used

JavaScript/TypeScript Firebase Docker React Figma Node.js GitHub

Key Features

  • Interactive dashboard with a map view showing fog net locations and associated water collection data
  • Customizable comparison tool to evaluate sites based on key factors (e.g., geography, climate, water yield)
