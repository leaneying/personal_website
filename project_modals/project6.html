Ocaml Scrabble

GitHub

Description

A command-line recreation of the classic word game Scrabble, fully implemented in functional programming using OCaml. This project emphasized modular design, recursive data structures, and test-driven development, applying core principles of functional programming to build a robust and interactive gameplay experience.

Date: August - December 2023
Company/Organization: Cornell University↗
Role: Full-stack developer

Technologies Used

Ocaml Dune Ubuntu GitHub

Key Features

  • Supports both single-player and two-player modes
  • Easy and hard difficulty settings ( Easy mode offers a list of all valid words that can be made from the player’s current tiles)
  • Full implementation of Scrabble rules, including double/triple word and letter scoring
  • Automatic score calculation and turn-based play tracking
